# Preguntas_Interesantes
Una pagina para preguntas que puedes hacer para hacer conversación 

Sitio web con 200 preguntas originales para iniciar conversaciones en citas, con amigos, familia y más.

# Características
- 200 preguntas
- 20 categorías diferentes
- Sistema de favoritos con localStorage
- Diseño responsive
- Efectos flip en tarjetas
- Búsqueda en tiempo real

# Tecnologías
- HTML5
- CSS3
- JavaScript vanilla
- Font Awesome
- Google Fonts
